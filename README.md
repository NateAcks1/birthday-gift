Romantic Birthday Surprise Website

A romantic "Happy Birthday" surprise website built using HTML, CSS, and JavaScript. When the correct date (11 Nov 2006) is entered, the page unlocks a full celebration with animations, a love story, and a heartfelt message.

Features:

Unlock screen that reveals the surprise after entering the correct date

Interactive birthday cake with a "Make a Wish" button

Three love story cards: From the First Day, Growing Together, Forever & Always

Responsive photo gallery with hover effects

Love letter with a typewriter animation

Romantic pink theme with hearts, confetti, and smooth transitions

Technologies:

HTML5

CSS3

JavaScript (Vanilla)

Google Fonts: Dancing Script and Poppins

Structure:
index.html
style.css
script.js
images/

Customization:

Change the unlock date in script.js

Replace gallery images in the images folder

Edit texts and messages in index.html

No frameworks were used. The design is fully responsive and clearly commented for easy editing.
